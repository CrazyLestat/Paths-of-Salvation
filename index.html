Make use of the right technologies

NFT 721

ERC-721 defines a minimum interface a smart contract must implement to allow unique tokens to be managed, owned, and traded in the Ethereum blockchain
NFT 1155

A smart contract on our platform is based on ERC-1155, which combines fungible and non-fungible (ERC-721) tokens. By using ERC-1155, you increase security and decrease redundant bytecode on Polygon. The Wolfs NFTs will be stored as an ERC-1155 token on Polygon blockchain and it is hosted on IPFS.
